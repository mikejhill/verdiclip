# default_config.json

This file is a **reference copy** of VerdiClip's default configuration. It is not loaded by the application at runtime — the canonical defaults are defined in `src/verdiclip/config.py` (`DEFAULT_CONFIG` dict).

This file serves as documentation for users and developers who want to understand the available configuration keys and their default values without reading source code.

If you modify defaults, update both `config.py` and this file to keep them in sync.
